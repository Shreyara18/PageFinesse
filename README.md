# 📩 PageFinesse

PageFinesse is a responsive landing page built using **HTML** and **CSS**. The page promotes a daily design inspiration newsletter where users can subscribe by entering their email address. It focuses on creating a clean, modern, and visually appealing user experience.

## 🌐 Live Demo

🔗 https://shreyara18.github.io/PageFinesse/

---

## ✨ Features

- 📧 Newsletter subscription interface
- 🎨 Modern and clean landing page design
- 📱 Responsive layout
- 🖼️ Hero illustration with promotional content
- 🔒 User-friendly email input section
- 📢 Clear call-to-action for registration
- 🔗 Social media links in the footer

---

## 🛠️ Tech Stack

- HTML5
- CSS3

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Shreyara18/PageFinesse.git
```

### Navigate to the project directory

```bash
cd PageFinesse
```

### Run the project

Simply open **index.html** in your preferred web browser.

For a better development experience, you can also use **VS Code Live Server**.
